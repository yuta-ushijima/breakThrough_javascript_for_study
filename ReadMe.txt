●一部のサンプルデータは、動作確認時にサーバにアップしてください
Chapter-04の全サンプル、Chapter-05のサンプル「chp05-04/03/」「chp05-04/04/」 「chp05-05/01/」「chp05-05/02/」「chp05-05/03/」は、ローカル環境では動作しないことがあります。サーバあるいはローカルサーバにアップして動作確認してください。

●Internet Explorerで動作確認する場合
ローカル環境でサンプルデータを開く際、Internet Explorerのセキュリティの設定内容によっては、以下のような警告が表示され、サンプルデータが表示されないことがあります。


＜警告例＞
セキュリティ保護のため、コンピュータにアクセスできるアクティブコンテンツは表示されないよう、Internet Explorerで制限されています。オプションを表示するには、ここをクリックしてください。


サンプルデータを表示できるようにするには、以下の方法があります。
※以下の作業を行った際に発生した障害やデータ破損などについては、著者および翔泳社は一切の責任を負いません。自己責任でお願いします。

＜方法1：オプションを表示して許可する＞
Internet Explorer上に表示された警告をクリックして、［ブロックされているコンテンツを許可］を選びます。すると、［セキュリティの警告］ダイアログボックスが開くので、［はい］を選んでアクティブコンテンツの実行を許可します。


＜方法2：インターネットオプションを変更する＞
警告が表示されないようにするは、Internet Explorerの［インターネットオプション］を開き、［詳細設定］タブにある［マイコンピューターのファイルでのアクティブコンテンツの実行を許可する］にチェックを入れます。
※サンプルデータの動作確認後は元に設定に戻すようにしてください。
